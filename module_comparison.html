Subject Teacher vs Class Teacher Module Comparison

Access Subject Teacher Dashboard Access Class Teacher Dashboard (Reference)

Subject Teacher Module

Purpose: Manages attendance for a specific subject across multiple classes
Scope: Subject-focused (e.g., Mathematics, English, Science)
Students: All students taking the specific subject
Attendance: Subject-specific attendance tracking
Database Tables:
  • tblsubjectteachers
  • tblsubjectteacher_student
  • tblsubjectattendance
  • tblsubjects

Class Teacher Module

Purpose: Manages attendance for a specific class/section
Scope: Class-focused (e.g., Class 10-A, Class 9-B)
Students: All students in the assigned class
Attendance: General class attendance tracking
Database Tables:
  • tblclassteacher
  • tblclass
  • tblclassarms
  • tblattendance

Key Features Comparison

Feature Subject Teacher Class Teacher
Take Attendance ✅ For specific subject ✅ For entire class
View Students ✅ Students taking the subject ✅ Students in the class
Attendance Reports ✅ Subject-specific reports ✅ Class-wide reports
Date Range Reports ✅ Available ✅ Available
Download Records ✅ Subject attendance data ✅ Class attendance data
